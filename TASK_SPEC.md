# Locale Dictionaries — Task Specification

## Опис
Односторінковий довідник локалей світу. Показує таблицю локалей з можливістю пошуку та перегляду детальної інформації в модальному вікні.

## Дані
Кожна локаль містить:
- **Код локалі** — `{iso_639_1}-{iso_3166_1_alpha2}` (напр. `en-AU`)
- **Мова** — назва мови (English)
- **Країна** — назва країни (Australia)
- **Валюта** — код валюти (AUD)
- **TLD** — домен верхнього рівня (.au)
- **Прапор** — емодзі-прапор країни (🇦🇺)
- **Столиця** — назва столиці (Canberra)
- **Таймзона** — часова зона (UTC+10:00)
- **Континент** — континент (Oceania)
- **Регіон** — регіон (Australia and New Zealand)
- **Площа** — площа країни (7,692,024 km²)
- **Сусідні країни** — список сусідів

## Стек
- **Backend:** Node.js (http module, без Express)
- **Frontend:** чистий HTML/CSS/JS (без frameworks)
- **Залежності:** жодних npm-пакетів

## Інтерфейс
- Мова UI: українська / англійська (перемикання)
- Дані локалей: завжди англійською

## Функціонал

### Таблиця
Колонки: прапор, код локалі, мова, країна, валюта, TLD, кнопка "Деталі"

### Пошук
Фільтрація в реальному часі (регістронезалежний contains) за полями:
- `locale` — код локалі
- `language.name` — назва мови
- `country.name` — назва країни
- `country.name_local` — локальна назва країни
- `country.currency_code` — код валюти
- `country.capital_name` — назва столиці

### Модалка "Деталі"
При кліку на кнопку в таблиці — модальне вікно з повною інформацією про локаль:
- Прапор (великий), код локалі
- Мова, країна, столиця
- Валюта (код + назва + символ), TLD
- Таймзона, континент, регіон
- Площа країни, сусідні країни (з прапорами)

### Стани UI
- **Loading:** skeleton-рядки поки дані завантажуються
- **Error:** повідомлення якщо дані не завантажились
- **Empty:** "Нічого не знайдено" якщо пошук не дав результатів
- **Normal:** відображення даних
