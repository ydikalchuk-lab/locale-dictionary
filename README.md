# Locale Dictionaries

Односторінковий довідник локалей світу. Таблиця з 377 локалями, пошук у реальному часі, детальна інформація в модальному вікні. Інтерфейс українською та англійською.

## Як запустити

1. Встанови [Node.js](https://nodejs.org/) (версія 18+)
2. Склонуй репозиторій
3. Запусти сервер:

```bash
node server.js
```

4. Відкрий у браузері: [http://localhost:3000](http://localhost:3000)

## Можливості

- **Таблиця локалей**: прапор, код, мова, країна, валюта, TLD
- **Пошук**: фільтрація в реальному часі за кодом, мовою, країною, валютою, столицею
- **Деталі**: клік на "Деталі" — модальне вікно з повною інформацією (столиця, таймзона, континент, регіон, площа, сусідні країни)
- **Мова інтерфейсу**: EN / UA (перемикання у шапці)

## Структура проєкту

```
LocaleDictionaries/
├── server.js          # Node.js HTTP-сервер (без залежностей)
├── locales.json       # Дані: 377 локалей
├── TASK_SPEC.md       # Специфікація проєкту
├── HANDOFF_NOTE.md    # Нотатка для рев'юера
├── milestones/        # Плани кожного етапу
├── public/
│   ├── index.html     # Головна сторінка
│   ├── style.css      # Стилі
│   ├── app.js         # Клієнтська логіка
│   └── i18n.js        # Переклади EN/UA
└── README.md
```

## API

| Шлях | Опис |
|---|---|
| `/api/locales` | JSON-масив усіх локалей |
| `/` | Головна сторінка |
| `/public/*` | Статичні файли |

## Стек

- **Backend:** Node.js (http module, без Express та npm)
- **Frontend:** чистий HTML / CSS / JavaScript
- **Залежності:** жодних
