# Locale Dictionaries — план реалізації

## Стек
- Node.js (http module, без Express/npm)
- Чистий HTML/CSS/JS (без frameworks)
- Дані: існуючий `locales.json`

## Файли

```
LocaleDictionaries/
├── server.js              # Node HTTP-сервер
├── locales.json            # Дані (існує, ~2MB)
├── public/
│   ├── index.html          # Головна сторінка
│   ├── style.css           # Стилі (таблиця + модалка)
│   └── app.js              # Клієнтська логіка
├── i18n/
│   └── translations.js     # EN/UA переклади для UI
└── README.md               # Інструкція
```

## Поведінка

### Сторінка
- Таблиця з колонками: прапор, код, мова, країна, валюта, TLD, кнопка "ℹ️ Деталі"
- Пошук над таблицею — фільтрація в реальному часі за полями: locale, language.name, country.name, country.name_local, country.currency_code, country.capital_name
- Перемикач мови інтерфейсу EN/UA (самі локалі — завжди англійською)
- Індикатор завантаження (skeleton)
- Повідомлення при помилці або пустих результатах

### Модалка
- Відкривається при кліку "Деталі"
- Затемнений фон, закриття по Escape / кліку поза карткою / кнопці ✕
- Деталі: прапор (великий), код, мова, країна, столиця, валюта, TLD, таймзона, континент, регіон, площа, сусідні країни

### Сервер
- GET `/` → `public/index.html`
- GET `/api/locales` → `locales.json`
- GET `/public/*` → статика
- 404 для всього іншого
- Порт 3000

## Milestones

### M1: Сервер + статика
Node-сервер, віддає locales.json, HTML, CSS, JS. locale не трансформується — віддається як є.

### M2: Фронтенд — таблиця + пошук + переклад
HTML/CSS/JS: таблиця з усіма локалями, пошук, EN/UA перемикання.

### M3: Модалка + фінальний полірунок
Модалка з деталями, індикатор завантаження, обробка помилок, README.

## Ключові угоди
