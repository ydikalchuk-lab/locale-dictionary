# M1: Сервер + роздача даних

## Завдання
Створити Node.js HTTP-сервер, який віддає статичні файли та дані locales.json.

## Кроки
1. Створити `server.js`:
   - HTTP-сервер на порту 3000
   - Роутінг: `/` → index.html, `/api/locales` → locales.json, `/public/*` → статика
   - Правильні MIME-типи (text/html, application/json, text/css, application/javascript)
   - 404 для невідомих шляхів
2. Налаштувати заголовки CORS (не обов'язково, але для зручності)
3. Додати базовий `public/index.html` — заглушка ("Loading..." з посиланням на JS)
4. Додати `public/i18n.js` — об'єкт перекладів EN/UA
5. Додати `public/app.js` — заглушка (консоль-лог)
6. Додати `public/style.css` — заглушка (базові стилі для body)

## Фініш
```bash
node server.js &
curl http://localhost:3000/api/locales | head -c 200
# → [{ "locale": "aa-ER", ... }]
curl http://localhost:3000/public/app.js
# → console.log(...) або underfined поки що
curl http://localhost:3000/nonexistent
# → 404 Not Found
```
Сервер не падає, всі три маршрути працюють.
