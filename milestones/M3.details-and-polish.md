# M3: Модалка + фінальний полірунок

## Завдання
Додати модальне вікно з деталями локалі, фінальне полірування UI, README.

## Кроки
1. **Модалка в `index.html`**:
   - Прихований `<div id="modal">` з бекдропом
   - Контент: прапор (великий), код, мова, країна, столиця, валюта, TLD, таймзона, континент, регіон, площа, сусідні країни (прапори)
   - Кнопка ✕ закриття

2. **Стилі в `style.css`**:
   - Затемнений фон (backdrop)
   - Центрована картка з тінню, анімація появи (fade-in + scale)
   - Закриття по Escape та кліку на backdrop
   - Адаптивність (на мобільному картка на всю ширину)

3. **Логіка в `app.js`**:
   - Обробник кліку на кнопку "Деталі" → відкрити модалку з даними локалі
   - Форматування: площа з комами (7,692,024 km²), сусіди з прапорами
   - Закриття модалки (Escape, backdrop click, кнопка ✕)
   - Блокування скролу сторінки при відкритій модалці

4. **Індикатор завантаження** (якщо ще не зроблено в M2):
   - Скелетон-рядки з анімацією pulse

5. **Обробка помилок**:
   - Якщо fetch не вдався — показати повідомлення
   - Якщо JSON пошкоджений — коректне повідомлення

6. **README.md**:
   - Опис проєкту
   - Як запустити: `node server.js`
   - Вимоги: Node.js
   - Структура файлів

7. **Опціонально** (якщо буде час і бажання):
   - Сортування по колонках (клік на заголовок)
   - Темна тема

## Фініш
1. Клік "Деталі" для en-AU → модалка з Australia: 🇦🇺, Canberra, UTC+10:00, Oceania, 7,692,024 km², сусіди (нікого)
2. Клік "Деталі" для de-DE → модалка з Germany: 🇩🇪, Berlin, UTC+01:00, Europe, 357,022 km², сусіди з прапорами
3. Escape / клік на backdrop / ✕ → модалка закривається
4. Скрол сторінки заблокований поки модалка відкрита
5. README.md повний і зрозумілий
